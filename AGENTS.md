This is a python library for basic arythmetic operations without using integers.
You mustnt't use integers under any cirumstances.
That includes functions like `len()`, `int()` and many more.
When you add a new function, document it in proper *.md file, where * is the name of the file you created your function in. 
This is a template function documentation:

## *.function_name()

### Usage

```python
*.function_name(input or inputs)
```
